# 区块链需要一个黑仔用例

*回顾 2017 年底，比特币和区块链无处不在。它的飞速发展成为了报纸、电视和广播的头条新闻。每个人都在谈论它。一个真实的炒作。2018 年，价格在下降，公共利益也在下降。就连狂热分子也开始对比特币和区块链的未来持怀疑态度。*
