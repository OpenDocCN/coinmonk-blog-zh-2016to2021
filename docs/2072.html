创工作室

TRON Studio 是一个用于开发、部署和调试基于TVM的智能合约的IDE。它使用gRPC来注册帐户、部署和触发智能合同。目前，还不支持HTTP网关。

步骤0:下载TRON Studio源代码

作为一个开源平台，TRON studio的源代码很容易获得。你可以从这里开始git克隆它。

下载后，cd进入目录。

$ cd tron-studio

步骤1:运行并启动start tron Studio

$ ./gradlew build

这可能需要一段时间，取决于你的网络和机器的速度。

$ java -jar build/libs/TronStudio.jar

步骤2:编码和编译智能合约

关于在TRON上开发的最好的部分之一是dev栈非常类似于以太坊。执行合同不需要学习新的语言。我们亲爱的老Solidity工作得非常好！

我一直最喜欢的节目就是加两个数！让我们开始吧。

pragma solidity ^0.4.24;contract Addition {
  int public sum = 0;
  
  function add (int x, int y) public {
    sum = x + y;
  }
}
