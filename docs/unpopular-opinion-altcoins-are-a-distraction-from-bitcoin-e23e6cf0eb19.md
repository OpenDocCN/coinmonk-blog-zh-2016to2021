# 冷门观点:替代币是对比特币的干扰

> 原文：<https://medium.com/coinmonks/unpopular-opinion-altcoins-are-a-distraction-from-bitcoin-e23e6cf0eb19?source=collection_archive---------3----------------------->

一万奥特币。这让我想起了 1997 年至 2000 年的网络热潮。我知道，我知道。他们有“效用”它们有用途和应用。有些人确实如此。一些网络初创公司也是如此。然而，活下来的人并不多。

我数不清这种事在我身上发生了多少次。我问某人，“嘿，你听说过比特币吗？”我总是得到这样的回答，“哦，我的朋友买了一些 Dogecoin。”
