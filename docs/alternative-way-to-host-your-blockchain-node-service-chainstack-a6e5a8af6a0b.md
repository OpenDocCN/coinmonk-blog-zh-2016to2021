# 我应该运行自己的节点吗？

在开发了几个项目之后，我发现基本上每个区块链项目还是需要至少一个节点来保证项目服务的稳定性。虽然以太坊有 infura，比特币有 bitcore，但它们目前不收费，这也意味着当服务出现问题时，它们不会提供任何服务、支持或及时处理。
