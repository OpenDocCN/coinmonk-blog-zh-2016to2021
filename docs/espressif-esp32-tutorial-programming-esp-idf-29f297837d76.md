## 使用 ESP-IDF 编程

ESP-32 的本地软件开发框架被称为 Espressif 物联网开发框架(ESP-IDF)。如果你想要最大的功能和兼容性，这是你应该使用的。它比其他一些编程选项更“硬核”,但它确实有助于您更好地理解幕后发生的事情。

ESP-IDF 功能包括基于菜单的配置、编译和将固件下载到 ESP32 板。
