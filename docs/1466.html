不管是什么原因让你记录你的代码，你很快就会发现写文档是非常乏味和耗时的，因为每个函数都有大量的样板文本。这就是为什么我们做了简单的Atomplugin—solidity—comments。它包含通过按下以下组合键为您的Solidity smart contract生成所有样板文件的基本功能:Ctrl + Alt + G.

例如，如果您有以下代码:

pragma solidity 0.4.24;contract SomeContract {function test(uint _param1, uint _param2) external {
        require(_param1 == 1 && _param2 == 2);
    }}

而你按下Ctrl + Alt + G，你会得到:

pragma solidity 0.4.24;/// @title SomeContract
/// @notice
/// @dev
contract SomeContract {    /// @notice
    /// @dev
    /// @param _param1
    /// @param _param2
    /// @return
    function test(uint _param1, uint _param2) external {
        require(_param1 == 1 && _param2 == 2);
    }}

然后你只需要填空。很酷，对吧？
