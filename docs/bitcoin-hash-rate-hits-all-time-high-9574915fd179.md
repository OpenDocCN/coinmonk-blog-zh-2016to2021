# 比特币散列率创历史新高

> 原文：<https://medium.com/coinmonks/bitcoin-hash-rate-hits-all-time-high-9574915fd179?source=collection_archive---------9----------------------->

当中国在 7 月禁止比特币挖矿时，比特币的全球杂凑率暴跌，跌幅超过 50%。换句话说，一半的比特币开采都在中国进行，突然间，矿工们不得不寻找新的家园。
