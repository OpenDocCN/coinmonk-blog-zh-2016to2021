## **想象一下**

一个开放的去中心化的互联网，鼓励用户参与，创作者拥有他们使用的平台的一部分。他们还可以在涉及这些平台未来的决策中拥有发言权，他们的大部分生计都来自这些平台。所有的进入门槛都是最低的。

## ***畅想 web 3.0***

**Web 3 及其前身**
在深入 Web 3 之前，让我们先来看看 Web 已经存在的变体，它们为这一演变奠定了基础。

## Web 1.0(20 世纪 90 年代到 21 世纪初)

第一代互联网被称为 Web 1.0。这是一个静态或只读的网站。它主要是只读的，内容贡献很少，没有用户交互。人们每天都无法在这个版本的网络上发布和贡献内容。只有大公司可以。它更多的是把人们和信息联系起来。

## **Web 2.0**(2000 年初至今)

**社交或互动网络**。脸书和 Youtube 等社交网络以及用户生成的内容蓬勃发展。支持用户参与和互动。以社区为重点，因为人与人之间的联系成为优先事项。web 2 的准入门槛很低，仅仅是你的 Whatsapp 身份或 youtube 视频，它让创造者经济变得更容易进入。应验甚至超越了比尔盖茨对互联网的一部分预言。他在 1996 年发表的这篇文章的部分内容是这样的。

互联网令人兴奋的一点是，任何人只要有一台个人电脑和一个调制解调器，就可以发布他们创作的任何内容。

## **Web 3.0** (开发中)

尽管 web 2 确实有利于创造者经济。降低进入门槛，允许更多的用户参与和互动。它确实有缺陷。 ***经济中没有权力平衡*** 。平台掌握着所有的权力并作出专制的决定，而创造者总是受他们以平台为中心的决定的支配。这些决定优先考虑盈利而不是授权创造者。

## **Web 3.0 到底是什么？**

这是正在发展的第三次互联网浪潮，应用程序和网站将以非常智能的方式分析数据。跨越我们大部分(如果不是全部)日常活动的数据将由机器处理。这是最接近于由创立 www - Tim Berners Lee 的人提出的最初的互联网去中心化计划的实现。他称之为语义网。

更有趣的是，它不会像 web 2 那样由中央集权的巨头统治，而是由用户来统治，他们每个人都可以通过对平台的发展做出贡献来拥有这些平台的一部分。这种模型的一个例子是[镜子。](https://mirror.xyz/race)镜像上的$WRITE 令牌允许平台上的写入者成为镜像道的一部分。然后，道决定了平台的发展。
值得注意的是，数据将会以一种更加分散的方式存储，使用开源协议，如加密本地区块链。

## **web 3 的四个改变游戏规则的特性**

1.  **NFTS** : [NFTs](/coinmonks/what-the-heck-are-nfts-dfd14e57200f?source=user_profile---------1-------------------------------) 引入了一种*数字稀缺性和独特性，旨在将权力还给创作者。在过去，创造者试图采用直接货币化来创造稀缺。使用电子书、专辑甚至订阅。主要的缺点是非法复制和分发这些资源。这就是 NFT 的用武之地。这项技术及其可验证的链上记录使创作者能够最大限度地利用少数 [100 名真正粉丝](http://li-jin.co/2020/02/19/100-true-fans/)而不是数百万粉丝的激情。然后，这 100 名真正的粉丝可以获得数字吹嘘权和其他形式的独家访问权。而大众则继续享受几乎免费的内容。已经在音乐行业推动这种模式的平台的一个例子是[目录](https://beta.catalog.works/)。*
2.  社交代币:社交代币将使创作者能够控制他们与粉丝的关系，鼓励更直接的参与。另一方面，粉丝们会认为支持艺术家不是一种帮助，而是一种投资。这些粉丝会尽他们所能帮助创作者成功。因为创作者的成功将决定代币的成功，从而为持有代币的粉丝群体带来利润。
3.  **数据所有权和可移植性**:当前的 web 2 结构让创作者在试图离开平台甚至被平台本身去平台化时不知所措。这将意味着把他们的观众留在这些平台上。substack 等以未来为中心的平台已经开始让创作者完全拥有他们的观众。Substack 允许作者在决定离开平台时，携带订阅读者的电子邮件列表。
4.  **DAOs -社区所有权和决策** : [DAOs](https://en.wikipedia.org/wiki/Decentralized_autonomous_organization) -分散的自治组织是以社区为中心的平台的完美典范。它们是由成员拥有和运营的在线社区。会员资格主要是通过社区令牌获得的。将这一点带入创造者经济将意味着关于算法、货币化甚至在这些平台上开发什么功能的决策将由社区共同做出。以前由以利润为导向的平台独自做出的决定。如前所述，这种模型的一个完美例子是[镜子。](https://mirror.xyz/race)

## **未来的平台**

web 3 革命不会由一家公司来领导，而是由那些分散决策、为创作者激活数据可移植性并激活其平台上令牌化媒介的公司来领导。而 web 3 运动的反对者会声称 web 2 巨头目前拥有站在他们一边的用户。我会说这只是时间问题。要么这些 web 2 巨头逐渐去中心化，采用这些模型。或者准备好与即将到来的以未来为中心的平台浪潮争夺一席之地。一项又一项政策，战线越拉越长。
