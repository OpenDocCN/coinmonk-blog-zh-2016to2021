# 区块链:什么是节点和主节点？

最近在区块链，节点和主节点是一个越来越受关注的话题。这是理所当然的，因为节点是区块链基础设施的重要组成部分。如果没有节点，区块链的数据将无法访问。你可以说节点*就是区块链*。
