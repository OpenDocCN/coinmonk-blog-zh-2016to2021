# 近期(1-3 年)的下一步是什么？

虽然区块链项目本身的当前状态看起来非常有前途，但技术还不是很成熟也是事实。到目前为止，我们看到的大多数应用程序都是特定于应用程序的，并且非常专注于特定的行业/问题。然而，在不久的将来，我确实看到了以下几个重要的焦点转移:

# 企业就绪区块链

在目前的形式下，区块链仍然被视为一种比企业希望的更透明的技术，并且比他们当前的解决方案/应用程序更不可扩展。为了确保区块链为企业做好准备，需要在标准、测试、集成、文档、隐私、安全和可扩展性方面做大量工作。

下图显示，到 2024 年，企业业务将达到 160 亿美元左右，CAGR 为 28%:

![](img/d5613505c00429e2d9479ff40e8b5334.png)

为了实现这一增长水平，未来 2-3 年在企业领域将非常重要，这将催生许多令人兴奋的解决方案。

# 私人区块链和混合区块链

私有分布式账本及其与公共区块链的结合肯定会有所增长。人们肯定会从云技术中获得并行性，并提出适合特定业务需求的解决方案。像 [Hyperledger](https://www.hyperledger.org/) 和 [Corda](https://docs.corda.net/) 这样的项目已经是朝着这个方向迈出的一大步。

# 增加研究兴趣和采用

正因为区块链承诺提高效率、降低成本和增加透明度，学术界和商界都渴望在 R&D 投资，并成为这一旅程早期的一部分。区块链的密码学、共识机制、性能、可扩展性、隐私性等局限性都是研究的重点。

在过去的 1 年中，有许多概念证明被转移到试点项目中。在接下来的几年里，看到一些试点项目取得成功并成为主流似乎是令人兴奋的。

# 联盟、标准化和联盟

区块链作为一项技术仍处于探索模式，工具、流程和标准化都在定义和开发中。虽然开发人员的快速入职需要工具和流程，但标准化将使业务更容易与各种现有的和新的系统集成。

像 [R3](https://www.r3.com/) 和 [Hyperledger](https://www.hyperledger.org/) 这样的联盟是这个方向的伟大创举。除了标准化，它们还可以影响和解决法规、安全、隐私和互操作性问题。

为了使这项技术能够大规模使用，人们正在努力提高互操作性。例如，利用比特币 UTXO 机制的 Qtum 是一种区块链，它与以太坊和比特币兼容。在不久的将来会有更多这样的努力。

# 技术融合

此时此刻，重大的技术变革正在发生。虽然区块链是其中一个重要的领域，但物联网、人工智能、AR/VR/MR 领域的进展将补充这一方向的努力。

为了让物联网变得极其有用，由多个组织拥有和管理的多个网络之间发生的数据交易必须确保所需的完整性、分散性和可扩展性。区块链的概念似乎非常适合确保个人数据不暴露于黑客攻击，网络中的薄弱环节得到及时识别和修复。使用基于有向非循环图的 [Tangle](https://www.youtube.com/watch?v=CZxH1V_zoug) 协议，该协议使用轻量级分布式账本， [IOTA](https://iota.org/) 已经在努力为物联网和机器经济中的许多用例提供可扩展性、量子电阻和去中心化。

同样，像 [Botchain](https://botchain.talla.com/) 这样的解决方案旨在确保企业自主代理的确定性和安全性，这是区块链将如何用于人工智能的一个很好的例子。

# 区块链即服务(BaaS)

作为一家数字化转型公司，看到区块链作为服务的潜力感觉棒极了。微软 Azure 已经在提供 BaaS，事实上，它们支持大多数常见的选项，如以太坊、Hyperledger Fabric 和 Corda。同样，IBM 在 Bluemix 平台上推出了他们自己的 [BaaS](https://www.ibm.com/blockchain/) ，使企业能够轻松地开始使用区块链。

电子政府即服务(EGaaS)是区块链组织经济、公共和社会活动的平台，这种服务将使我们能够打破官僚程序，从而消除腐败和挫折。

# 摘要

到目前为止，毫无疑问，区块链具有颠覆性，它将改变我们开展日常业务的方式。它已经在挑战许多现有企业的现状，人们肯定会提出提高效率和透明度的应用程序。所有这些都有希望让这个世界变得更美好。
