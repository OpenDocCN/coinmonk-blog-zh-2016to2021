我想生成一个指定的钱包地址，而不是一个随机的，使它更酷，能够识别，而使用它。例如以cat结尾

我找到了 vanieth ，你可以用它来做以太坊，但是它不能和 Binace链一起使用，所以我从stressStellar账户生成器中抓取了一些代码，它是由Rungsikorn Rungsikavanich创建的，并与币安链/go-sdk 混合来为我生成账户

TLDR；

underior g/vani bnb

一个用golang写的币安链虚位地址生成器。-underior g/vani bnb

github.com

装置

$ go get -u github.com/undefiorg/vanibnb

使用

  • -s:要查找的后缀
  • -n:网络 mainnet 或 testnet

奔跑

$ vanibnb -s=cat -n=testnet
