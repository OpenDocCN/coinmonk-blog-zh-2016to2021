[Lightning Network](https://lightning.network/)是一个用于即时、大批量小额支付的分散系统，消除了将资金托管委托给可信第三方的风险。

闪电网络是使用比特币内置脚本的多方智能合约(可编程货币)的首批实现之一。闪电网络正引领比特币多方金融计算的技术发展[1]。它将实现即时支付，即使是小额交易，并将提高比特币的可扩展性，从而有助于比特币的采用。
