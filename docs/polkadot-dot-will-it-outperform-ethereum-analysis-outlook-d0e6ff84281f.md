Polkadot token (DOT)价格在二月份与其他加密货币一起暴涨，自年初以来上涨了 353 %,达到 42 美元大关。
Polkadot 正在**变得受欢迎**，因为该项目旨在通过连接区块链并允许它们彼此交互来分散网络，这将实现**新的应用，如某些分散的金融** **服务** (DeFi)。
