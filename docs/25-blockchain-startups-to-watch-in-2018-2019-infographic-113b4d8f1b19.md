# 比特币和区块链可能正在超越流行语的地位

对该行业的投资从 2012 年的几乎为零，增加到过去两年每年超过 5 亿美元。企业及其风险部门，尤其是金融服务领域的企业，已经大量加入竞争，希望借助区块链和分布式总账解决方案来解决棘手问题，包括数据对账、清算和结算等。全球主要银行和金融中介机构正与区块链公司密切合作，利用新技术改造遗留系统和基础设施。
