下载并安装Docker平台

  1. 从 Docker for Windows下载网站下载Docker for Windows(稳定版)安装程序

2.在下载文件夹中，双击Docker for Windows Installer.exe可执行文件。

3.安装将开始。

4.安装完成后，将要求您注销并重新登录到您的计算机。(可能会要求您重新启动计算机。)

5.当你登录到你的电脑后，Docker将会启动。如果Docker通知没有出现，你可以加倍显示桌面上的Docker for Windows图标。

注意:显示在Windows任务栏右侧的Docker“whale”图标。

测试Docker平台

6.在Windows任务管理器中，您应该看到Docker进程正在运行

7.在Hyper-V管理器中，您应该看到MobyLinuxVM已经安装并正在运行。

8.注意:如果您运行的不是有效版本的Windows 10或者没有正确安装Hyper-V服务，Docker可能会显示一条与此类似的消息。咨询Windows网站的Docker以获得更多帮助。

9.通过启动Windows Powershell来测试Docker本身。从Windows“开始”菜单中，键入“powershell”。从列表中选择Windows Powershell应用程序。

10.在Powershell控制台中，输入docker run hello-world(并按回车键)。

11.Docker容器平台的安装和测试已经完成。
