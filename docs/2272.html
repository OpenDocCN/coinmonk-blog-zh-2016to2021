问题#1

protoc version is too old. Update protoc to 3.6.0 or above. Abort

解决方案#1

首先，检查你的协议版本

protoc --version

你应该得到…

libprotoc 3.0.0

这显然与 3.6.0 不匹配，所以让我们升级它。

在macOs上，你可以…

brew install protoc

没有啤酒吗？然后安装自制软件

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
