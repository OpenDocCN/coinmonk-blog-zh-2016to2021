# Hyperledger 结构-简介

![](img/82ae0ac6aa8af48d9dc8ade9e08d1809.png)

**区块链**是最近一年的热门话题。当你听到这个词时，你首先想到的可能是加密货币，如*比特币*或*虚拟货币*。但是区块链不仅仅局限于克里托币，还有各种用途无穷的区块链。例如，你可以使用区块链出售或出租汽车和房屋，管理慈善捐款等。
