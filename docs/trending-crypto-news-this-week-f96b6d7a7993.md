# 😮学生投资加密激增 3 倍

考虑到学生背负的沉重债务和每月高昂的生活费用，可能没有人会感到惊讶的是，越来越多的学生转向加密作为收入来源。

英国的一项调查发现，在过去的 12 个月里，对 2000 多名大学生的加密投资增加了两倍，从 2%增加到 6%。我们可以假设疫情在这方面发挥了作用，但毫无疑问，媒体对加密空间的更大关注为新兴一代打开了新的大门。
