Docker 是一个包装和运输代码的神奇工具。这种优势源于这样一个事实，即你可以将所有的**依赖项**打包到 Docker 文件中，然后你的应用就可以在任何有 Docker 的地方运行了。

另一个有许多依赖项的地方是开发环境。作为一个开发工具，Docker 并不那么出名，但是没有理由说它不能这样使用…
