# **BIP 一瞥——直根**

BIP，也被称为比特币改进提案，是比特币开发者用来进行几乎所有比特币相关技术改进和参数调整的形式。任何重量级的 BIP 都容易引发比特币开发者和整个社区的热烈讨论，甚至可能引发**分裂社区共识**……
