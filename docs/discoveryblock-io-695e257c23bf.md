# 区块链

**为什么？** Discovery architecture 使用区块链进行平台内的数据令牌化服务，这意味着信息提供商和消费者之间的每一次价值交换都被令牌化，以创建一个由令牌管理的动态、一致的协作生态系统。

由于优先级 n 1 允许一个价值(代币)按比例公平分配的微交易平台，我们正在实施新技术，允许以极低的费用进行即时交易。
