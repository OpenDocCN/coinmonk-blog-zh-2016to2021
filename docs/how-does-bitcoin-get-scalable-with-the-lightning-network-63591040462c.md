# 比特币如何通过闪电网络实现可扩展？

闪电网络是比特币区块链的离线解决方案。这是“第二层解决方案”有了这些技术，区块链的操作可以外包给外部系统处理。对于区块链，这种技术是最著名和最好的解决方案之一。
