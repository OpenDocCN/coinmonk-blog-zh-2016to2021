今天我们有一个类似的故事，有人变得疯狂👉钱。

AYFKM

马人·奥特曼是一个吉他手/占星大师/加密货币“不是一个财务顾问”/影响者。

实际上，我应该换一种说法:马人·奥特曼是占卜者/占星术专家/加密货币“不是财务顾问”/影响者。

她使用占星术作为指标来预测加密货币的价格走势。这是我有时也喜欢使用的许多指标之一，尤其是当我连续感冒的时候。

我的一切都想在这里取笑奥特曼(以及任何为此而看星座的人)。我想说:“嘿，你不觉得认为占星术(这只是伪科学的另一个名字)与处女崇拜的虚构数字符号有任何关系有点可笑吗！?"

但我不会这么说，因为我认为她可能是某种偶然的天才。我表现出的自制力值得称赞。

她拿走了两件让人彻底发疯的东西(占星术&密码)并把它们放在一口大小的夹子里，这样即使是患有多动症的黑猩猩也能消化它们。排序:

占星术是一个、2.2亿美元的行业，全球密码市值为~ 1.6亿美元(带T)。

委婉地说:那是一个狗屎吨的愚蠢的钱在两个非常……暧昧的行业里浮动。奥特曼的视频和“建议”也模棱两可。然而，她的野心却没有。

“我是一个真正认真的人。” - 马人奥特曼

她不是在开玩笑，她真的是认真的。

当她十几岁的时候，她在聚会上向喝醉的孩子出售占星术，仅仅是为了赚点钱。现在这位占星解读-加密-抽-嘀-Tok-影响力的精神病患者不知何故成功地在她的所有社交媒体账户上超过了百万粉丝。

这让她成为专家了吗？你告诉我！

这一点我不能否认。

奥特曼是不是专家无关紧要。与相关的是她成功地将这些痴迷变成了金钱。她利用自己庞大的追随者形成了几个利润丰厚的收入来源:

  • 月亮的奥秘 - $199 (抱歉，没有奖学金)
  • 密码图表 - $385 或每月$35
  • 网站会员资格:
  • 受益人-每月49.49美元
  • 天文倡导者-每月15.15美元
  • 杰出的爱人——每月7.77美元
  • 商品，书籍，播客，咨询

占星术、加密货币和Tik Tok。这是愚蠢的完美风暴。

这也是创造一个真正的自动提款机的秘诀，这个提款机由永无止境的哑钱(占星术+密码书呆子)定期补充。
